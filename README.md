# 📊 Análise de Campanha de Marketing

Dashboard interativo desenvolvido para analisar o desempenho de mais de 10.000 campanhas de marketing, transformando dados em insights estratégicos para apoio à tomada de decisão.

## 🎯 Objetivo

Analisar campanhas de marketing, identificar padrões de desempenho entre diferentes canais e gerar insights para otimização dos resultados.

## 📂 Dataset

Base fictícia com mais de 10.000 campanhas contendo informações de investimento, receita, cliques, impressões, conversões, canal de marketing e período da campanha.

## 📈 Indicadores Analisados

* Receita
* Investimento
* Lucro
* ROI
* CTR
* Taxa de Conversão
* CPA
* Conversões

## 🔍 Principais Insights

* **Search:** maior ROI médio entre os canais.
* **Display:** maior receita e lucro total.
* **Influencer:** menor CPA e alto volume de conversões.
* **Social:** maior taxa de conversão.

## 📊 Dashboard

O dashboard apresenta:
