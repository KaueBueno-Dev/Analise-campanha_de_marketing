# 📊 Análise de Campanha de Marketing

Dashboard interativo desenvolvido para analisar o desempenho de 10.000 campanhas de marketing, transformando dados em insights estratégicos para apoiar a tomada de decisão.

## 🎯 Objetivo

O objetivo deste projeto é explorar dados de campanhas de marketing, identificar padrões de desempenho entre diferentes canais e gerar recomendações de negócio com base em indicadores-chave de performance.

## 📂 Dataset

O conjunto de dados contém informações de 10.000 campanhas de marketing, incluindo:

- Canal de Marketing
- Investimento
- Receita Gerada
- Cliques
- Impressões
- Conversões
- Público-Alvo
- Localização
- Período da Campanha

## 📈 Indicadores Analisados

- 💰 Receita Total (Revenue)
- 💵 Investimento Total (Cost)
- 📊 Lucro Total (Profit)
- 🚀 ROI (Return on Investment)
- 🎯 CTR (Click Through Rate)
- 🔄 Taxa de Conversão
- 📉 CPA (Cost per Acquisition)
- ✅ Total de Conversões

## 🔍 Principais Insights

### 🔎 Search
- Apresentou o maior ROI médio entre todos os canais.
- Demonstrou maior eficiência na geração de retorno sobre investimento.

### 📺 Display
- Responsável pela maior receita total.
- Gerou o maior lucro acumulado.
- Canal com maior impacto financeiro para o negócio.

### 🤝 Influencer
- Menor CPA entre os canais analisados.
- Alto volume de conversões.
- Excelente custo-benefício para aquisição de clientes.

### 📱 Social
- Maior taxa de conversão.
- Necessita otimização para aumentar a rentabilidade das campanhas.

## 💡 Recomendações Estratégicas

- Aumentar investimentos em campanhas Search devido ao alto ROI.
- Expandir campanhas Influencer aproveitando o baixo CPA.
- Manter a estratégia atual em Display devido ao elevado volume de receita e lucro.
- Revisar campanhas com ROI muito baixo ou lucro reduzido.
- Otimizar campanhas Social para melhorar o retorno financeiro.
